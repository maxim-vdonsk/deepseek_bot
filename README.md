# DeepSeek Telegram Bot

Telegram-бот с поддержкой AI на базе [DeepSeek](https://platform.deepseek.com/).
Отвечает на текстовые вопросы и умеет анализировать загружённые документы (PDF, DOCX, TXT).

---

## Что умеет бот

- Отвечать на текстовые вопросы с помощью модели `deepseek-chat`
- Загружать и анализировать документы форматов **PDF**, **DOCX**, **TXT** (до 10 MB)
- Помнить историю разговора в рамках одной сессии (хранит контекст в SQLite)
- Автоматически повторять запрос при временных ошибках API (до 5 попыток)
- Записывать все события в файл `bot.log`

```

├── .env             # Ваши секретные ключи (НЕ коммитить в git!)
├── bot.log          # Логи (создаётся автоматически)
└── bot_context.db   # База данных (создаётся автоматически)
```

## Установка и запуск (шаг за шагом)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/maxim_vdonsk/deepseek_bot.git
cd deepseek_bot
```

### 2. Создать виртуальное окружение

Виртуальное окружение изолирует зависимости проекта от системного Python.

```bash
# Создаём окружение
python -m venv venv

# Активируем его:
# macOS / Linux:
source venv/bin/activate

# Windows:
venv\Scripts\activate
```

> Вы поймёте, что окружение активно, когда в терминале перед строкой появится `(venv)`.

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

Скопируйте файл-шаблон и заполните его своими данными:

```bash
cp .env.example .env
```

Откройте `.env` в любом редакторе и вставьте свои токены:

```env
TELEGRAM_BOT_TOKEN=123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
BASE_URL=https://api.deepseek.com/v1
```

> **Важно:** файл `.env` содержит секретные ключи — никогда не добавляйте его в git!
> Добавьте `.env` в `.gitignore`, если его там нет.

### 5. Запустить бота

```bash
python main.py
```

Бот запустится и начнёт принимать сообщения. Для остановки нажмите `Ctrl+C`.

## Как пользоваться ботом

| Действие | Что делает бот |
|---|---|
| `/start` | Сбрасывает историю диалога и приветствует вас |
| Отправить текст | Отвечает на вопрос с учётом истории разговора |
| Отправить файл (PDF/DOCX/TXT) | Извлекает текст и сохраняет его для последующих вопросов |

**Пример сессии:**
1. Отправьте `/start`
2. Загрузите файл `contract.pdf`
3. Напишите: _"Какой срок действия у этого договора?"_ — бот ответит, используя текст из файла

## Технические детали

| Параметр | Значение |
|---|---|
| Максимальный размер файла | 10 MB |
| Лимит контекста | 32 KB (при превышении — старые сообщения обрезаются) |
| Лимит сообщения Telegram | 4096 символов (длинные ответы разбиваются автоматически) |
| Повторных попыток при ошибке | 5 (с паузой 3 секунды) |
| База данных | SQLite (`bot_context.db`) |
